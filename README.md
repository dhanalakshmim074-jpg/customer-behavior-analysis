# Customer Shopping Behavior Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior using an end-to-end data analysis pipeline. The goal is to extract meaningful insights from raw data and present them through an interactive dashboard.

## Tools & Technologies Used
- Python (Pandas)
- MySQL
- Power BI
- Excel

## Project Workflow

### Data Collection
- Raw dataset collected from Excel file

### Data Cleaning (Python)
- Handled missing values
- Removed duplicates
- Data transformation using Pandas
- Prepared clean dataset for analysis

### Database Management (MySQL)
- Created database and tables
- Imported cleaned data into MySQL
- Structured data for efficient querying

### SQL Analysis
Performed multiple business queries such as:
- Revenue analysis by gender
- Average purchase behavior
- Discount impact analysis
- Customer segmentation using NTILE
- Top-performing product categories
- Subscription insights

### Data Visualization (Power BI)
- Built interactive dashboard
- Visualized key KPIs:
  - Total Revenue
  - Average Purchase
  - Customer Segments
  - Product Performance
- Used slicers and filters for dynamic analysis

## Key Insights
- Identified high-revenue customer segments
- Analyzed impact of discounts on purchases
- Found top-performing products and categories
- Evaluated subscription behavior trends
